## 🎯 Visión General
Este repositorio es el archivo central de mi formación como **Analista de Datos** en **EBAC**. Contiene la resolución de casos prácticos, desde los fundamentos en hojas de cálculo hasta la implementación de arquitecturas en la nube y pipelines de datos.

## 🛠️ Competencias Desarrolladas
* **Análisis de Datos:** EDA avanzado (`.ipynb`) para descubrir patrones en datasets.
* **Ingeniería de Datos:** Consultas SQL (`.sql`) y estructuras relacionales.
* **Visualización:** Reportes ejecutivos (`.pptx`, `.pdf`) con enfoque en Storytelling.
* **Infraestructura Cloud:** Implementación de pipelines y configuración de entornos (AWS/Cloud).
